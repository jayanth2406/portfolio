# Jayanth Portfolio

Personal portfolio website built with React, Vite, React Router, Tailwind CSS, and Framer Motion.

Live site: https://jayanth2406.github.io/portfolio/

## Stack

- React 19
- Vite 8
- React Router
- Tailwind CSS v3
- Framer Motion
- Lucide React

## Pages

- Home
- Projects
- About
- Contact

## Run locally

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Deploy to GitHub Pages

```bash
bun run deploy
```

The project is configured for repo-hosted Pages with:

- Vite base path: `/portfolio/`
- Hash-based routing for reliable client-side navigation on GitHub Pages
